# 🎮 模板系统使用指南

## 📍 访问地址

### 正式使用页面（用户界面）
```
http://localhost:3004/setup
```
这是你项目的**"开始使用"页面**，用户在这里输入关键词进行识别。

### 测试页面（调试用）
```
http://localhost:3004/test-template.html
```
这个页面会显示 `_isTemplate: true` 标记，方便你调试。

---

## 🎯 如何使用

### 步骤 1：访问 setup 页面
打开浏览器访问：`http://localhost:3004/setup`

### 步骤 2：选择输入方式
- **📷 图片上传**：上传物品图片
- **✏️ 文字输入**：直接输入物品名称（推荐测试模板）

### 步骤 3：输入关键词
在文字输入框中输入以下任意关键词：

#### 【大国重器】(4/5)
- ✅ "空间计算头显" / "AR头显" / "VR头显"
- ✅ "火箭发动机" / "液氧甲烷"
- ✅ "机器狗" / "四足机器人"
- ✅ "船舶推进器" / "吊舱推进"

#### 【极客定制】(3/5)
- ✅ "机械键盘" / "客制化键盘"
- ✅ "微单相机" / "无反相机"
- ✅ "碳纤维自行车" / "公路自行车"

#### 【生活黑科技】(3/5)
- ✅ "无叶吹风机" / "吹风机"
- ✅ "宠物喂食器" / "智能喂食器"
- ✅ "滑雪护目镜" / "AR护目镜"

#### 【绿色新消费】(4/5)
- ✅ "培育钻石" / "人造钻石"
- ✅ "人造肉" / "植物肉"
- ✅ "环保跑鞋" / "可降解鞋"
- ✅ "盲盒" / "潮玩"

### 步骤 4：点击"识别"按钮
系统会自动判断：
- **命中模板** → 1-1.5秒后显示预设数据
- **未命中** → 2-3秒后调用真实 AI

### 步骤 5：查看结果
你会看到：
- 📝 物品名称
- 🏷️ 分类标签
- 📄 简短描述
- 🎨 图标（emoji）
- 🖼️ 图片（从 Wikimedia 获取）
- 🚀 "开始拆解"按钮

### 步骤 6：开始拆解
点击"开始拆解"按钮，进入可视化拆解界面。

---

## 🔍 如何判断是否命中模板

### 方法 1：查看延迟时间
- **1-1.5秒** → 很可能是模板
- **2-3秒** → 很可能是真实 AI

### 方法 2：使用测试页面
访问 `http://localhost:3004/test-template.html`，会显示：
```json
{
  "_isTemplate": true,  // 命中模板
  "_templateKey": "mechanical-keyboard"
}
```

### 方法 3：查看浏览器控制台
打开开发者工具（F12），查看 Console：
- `🎯 精确匹配: "机械键盘" -> "客制化机械键盘"` → 命中模板
- `❌ 未命中模板: "手机"` → 调用真实 AI
