# 智能制造业多Agent系统

这是一个基于多Agent架构的智能制造业产品拆解与供应链分析系统。

## 功能特性

- **产品智能拆解**: AI自动将产品拆分为4-6个零件
- **多Agent协同**: 7个专业Agent协同工作
  - 供应商Agent: 检索符合规格的供应商
  - 定制化Agent: 引导用户定制零件
  - 工艺Agent: 生成可行的工艺方案
  - 成本Agent: 多维度成本分析
  - 工程风险Agent: 评估潜在风险
  - 碳排放Agent: 碳足迹评估
  - Keep Breaking Agent: 综合决策建议

- **实时思考展示**: 所有Agent的思考过程实时滚动显示
- **递归拆解**: 每个层级的零件都可以继续拆解
- **状态A评估**: 成本、风险、碳排放三维度并行评估

## 安装依赖

```bash
npm install
```

## 运行系统

```bash
npm start
```

访问 http://localhost:3000

## 运行测试

```bash
npm test
```

## 技术栈

- 后端: Node.js + Express + WebSocket
- 前端: 原生JavaScript + HTML5 + CSS3
- AI: DeepSeek API
- 测试: Playwright

## 使用流程

1. 输入产品名称（例如：白色金属闹钟）
2. 系统自动拆解为零件
3. 选择零件，查看供应商选项
4. 选择供应商或定制零件
5. 进入状态A，查看综合评估
6. 根据建议决定是否继续拆解

## 注意事项

- 所有Agent结果由AI生成，不使用硬编码
- 系统支持多层级递归拆解
- 每个零件都可以独立完成完整的分析循环
